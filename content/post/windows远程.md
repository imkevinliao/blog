# windows远程
因为有的时候需要别人帮忙处理或者帮别人处理一些电脑上的问题，所以需要用到windows的远程工具。自带的当让是最有解决了，但是体验是真的糟糕。

当然如果仅仅是体验糟还能忍受的话，那有的时候完全就是连不上的状态，那真让人抓狂。

# 关键字
远程工具的使用基本不要说的，应该是都会的。

`远程协助  命令行  msra`

`远程桌面连接`

# 第三方工具
但凡windows自带的可以勉强能用也不必使用第三方了。
